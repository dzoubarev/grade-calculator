Meant to help McGill Students calculate what they need on exams to get the grades they want. (And teach me some web dev)

Features:
Admin Dashboard,
Creating a course from scratch (similar to other grade calculator tools),
Feedback page,
Growing database of McGill courses and their grading schemes


Frontend: React + Material UI for styling

Backend: Java Springboot and PostgreSQL (Supabase)

Deployment: Vercel for frontend, and Render for Backend
